# konva-rs

Konva in Rust

### Quick Start

#### Run Test

```shell
$ cargo test
```

#### Run WASM

```shell
$ wasm-pack build wasm --scope rustq
```

```shell
$ cd wasm/playground
```

```shell
# nvm use 16
$ npm i;
$ npm run serve;
# then open http://localhost:8080/
```
